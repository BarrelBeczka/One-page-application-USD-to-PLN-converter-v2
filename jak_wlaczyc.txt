INSTRUKCJA URUCHOMIENIA PROJEKTU KROK PO KROKU
=================================================

KROK 1: URUCHOMIENIE BAZY DANYCH (PostgreSQL)
---------------------------------------------
1. Otwórz program pgAdmin 4 (lub użyj konsoli psql).
2. Upewnij się, że serwer PostgreSQL jest uruchomiony.
3. Sprawdź, czy masz utworzoną bazę danych o nazwie: currency_db
4. WAŻNE: Upewnij się, że hasło do bazy zgadza się z tym w kodzie aplikacji.
   - Otwórz plik: backend/app.py
   - W linijce 12 znajdziesz zmienną HASLO_DO_BAZY. Wpisz tam swoje aktualne hasło do PostgreSQL (np. z uczelni).

KROK 2: URUCHOMIENIE BACKENDU (Terminal nr 1)
---------------------------------------------
1. Otwórz w edytorze (np. VS Code) główny folder projektu.
2. Otwórz pierwszy terminal.
3. Wejdź do folderu backendu komendą:
   cd backend
4. Aktywuj środowisko wirtualne (jeśli nie istnieje, najpierw utwórz komendą: py -m venv venv):
   - W PowerShell: venv\Scripts\Activate.ps1
   - W CMD: venv\Scripts\activate.bat
5. Zainstaluj wymagane biblioteki (wymagane tylko za pierwszym razem):
   pip install -r requirements.txt
6. Uruchom serwer backendu:
   py app.py

   -> Serwer uruchomi się na porcie 8080 (pojawi się komunikat "Running on http://127.0.0.1:8080"). 
   -> Zostaw to okno w spokoju. NIE wyłączaj tego terminala. Niech serwer działa w tle!

KROK 3: URUCHOMIENIE FRONTENDU (Terminal nr 2)
----------------------------------------------
1. Otwórz DRUGI, zupełnie nowy terminal (np. przyciskiem '+' w VS Code).
2. Wejdź do folderu frontendu komendą:
   cd frontend
3. Zainstaluj pakiety Node.js (wymagane tylko za pierwszym razem):
   npm install
4. Uruchom aplikację na frontendzie:
   npm run dev
5. W terminalu pojawi się link (np. http://localhost:5173/ albo http://localhost:5174/). 
6. Przytrzymaj klawisz CTRL i kliknij lewym przyciskiem myszy w ten link. 
7. Aplikacja otworzy się w Twojej przeglądarce i wszystko powinno działać!
